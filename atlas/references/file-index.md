# `_index.md`

`_index.md` is the navigation file for a project's knowledge layer.

## Rules

- `# <Project Name> Index`
- `## Core Pages` is required and curated
- `## Knowledge Pages` is required and exhaustive
- one line per page
- line format:
  - `- [page-name](knowledge/page-name.md) — type — one-line summary`
- `Core Pages` should be intentionally ordered
- if there are no core pages yet, keep the heading and write `_No core pages yet._`
- `Knowledge Pages` should usually be alphabetical by page name
- keep `_index.md` concise and scannable
