# `_client.md`

`_client.md` is the stable relationship context file at the client level. It holds enduring client facts and links down to the relevant project layer.

## Backlinking

- Must link to all `_project.md` pages.

## Shape

- `# {Client Name} Context`
- `## Context` — relationship background, constraints, key contacts, and standing decisions.
- `## Projects` — links to all `_project.md` pages.

## Content rules

- Do not summarize live project operating status here.
- Current work, blockers, owners, and near-term next steps live in project `_state.md` files.
