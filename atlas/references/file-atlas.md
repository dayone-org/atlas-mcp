# `_atlas.md`

`_atlas.md` is the stable orientation file at the workspace root. It holds company-wide enduring facts and links down to the relevant client layer.

## Backlinking

- Must link to all `_client.md` pages.

## Shape

- `# {Company Name} Context`
- `## Context` — company-wide background, constraints, actors, and standing decisions.
- `## Clients` — links to all `_client.md` pages.

## Content rules

- Do not summarize client or project operating status here.
- Current work, blockers, owners, and near-term next steps live in project `_state.md` files.
